Component Options

columnWidth

columnWidth: number
  • Defined in src/ui/ImageResultList/ImageResultList.ts :50

Specifies the width of a column in a column layout. The default value is 170.

heightThreshold

heightThreshold: number
  • Defined in src/ui/ImageResultList/ImageResultList.ts :45

Specifies the maximum height of a row in a row layout. The default value is 250.

layoutType

layoutType: string
  • Defined in src/ui/ImageResultList/ImageResultList.ts :37

Specifies the type of layout used to display images results. The available values are: row - Displays resized images that fit to the width of the container and have the same height for a row. column - Displays the images in fixed-size columns. The default value is row.
