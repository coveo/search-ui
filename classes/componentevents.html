Class ComponentEvents

Used by the various Coveo Component to trigger and bind event.
It adds a small logic to execute handler or triggers only when the component is "enabled".
A component is disabled by calling {Component.disable}
Typically, a component is disabled when it is not active in the current {Tab}.
It can also be disabled by external code.
The class serves as a way to not execute handler on component that are invisible and inactive in the query.
