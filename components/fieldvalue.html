Coveo component FieldValue

This component can be used as part of a result template to display the value of a field associated with the current search result.
This component is usually located inside a FieldTable.
A common use of this component is to display a specific field value, when that field also happens to be a facet. When the field value is clicked on, the corresponding facet is activated.

displaySeparator

displaySeparator: string
  • Defined in src/ui/FieldTable/FieldValue.ts :90

Specifies the string used to display multi-value fields in the UI. It is inserted between the displayed values. The default value is , .

Markup configuration example(s) :
data-display-separator='foo'

facet

facet: string
  • Defined in src/ui/FieldTable/FieldValue.ts :67

Specifies the facet to be toggled when the component is clicked On.
When no value is specified, the value of the field option is used.
If the facet ID is custom (e.g. not the same name as its field), you must specify manually this option in order to link the correct facet.

Markup configuration example(s) :
data-facet='foo'

splitValues

splitValues: boolean
  • Defined in src/ui/FieldTable/FieldValue.ts :79

Specifies if the field value is to be split at each FieldValue.options.separator. This is useful for splitting groups by a facet field.
The values displayed are split by the FieldValue.options.displaySeparator.
The default value is false.

Markup configuration example(s) :
data-split-values='true'
data-split-values='false'
