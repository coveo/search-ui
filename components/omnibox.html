Coveo component Omnibox

This component is very similar to the simpler Querybox component and support all the same options/behavior except for the search-as-you-type feature.
In addition, it takes care of adding a type-ahead capability. The type-ahead and the suggestions it displays are customizable and extensible by any custom component.
The type-ahead is configurable by activating addon which are provided OOTB (facets, analytics suggestions, Reveal suggestions, and advanced Coveo syntax suggestions).
It is also possible for external code to provide suggestions.

Component Options

enableFieldAddon

enableFieldAddon: boolean
  • Defined in src/ui/Omnibox/Omnibox.ts :99

Specifies whether the field addon should be enabled.
The field addon allows the search box to highlight and complete field syntax.
Default value is false.

Example: You want to filter on a file type. You start typing @sysf and matching fields are proposed. You select the @sysfiletype suggestion, enter = and the available matching file types are proposed.

Markup configuration example(s) :
data-enable-field-addon='true'
data-enable-field-addon='false'

enableQueryExtensionAddon

enableQueryExtensionAddon: boolean
  • Defined in src/ui/Omnibox/Omnibox.ts :113

Specifies whether the query extension addon should be enabled.

enableRevealQuerySuggestAddon: boolean

enableSearchAsYouType

enableSearchAsYouType: boolean
  • Defined in src/ui/Omnibox/Omnibox.ts :81

Specifies whether a new query is automatically triggered whenever the user types new text inside the query box.
Activate this as well a inline + Reveal suggestions for a cool effect!
The default is false.

Markup configuration example(s) :
data-enable-search-as-you-type='true'
data-enable-search-as-you-type='false'

inline

inline: boolean
  • Defined in src/ui/Omnibox/Omnibox.ts :75

Specifies that suggestions appearing in the omnibox should push the result down, instead of appearing over the results.
Activate this as well a SearchAsYouType + Reveal suggestions for a cool effect!
Default value is false

Markup configuration example(s) :
data-inline='true'
data-inline='false'

omniboxTimeout

omniboxTimeout: number
  • Defined in src/ui/Omnibox/Omnibox.ts :122

Specifies a timeout before rejecting suggestions in the omnibox.
Default value is 2000 (2 seconds).

Markup configuration example(s) :
data-omnibox-timeout='10'

searchAsYouTypeDelay

searchAsYouTypeDelay: number
  • Defined in src/ui/Omnibox/Omnibox.ts :86

When search as you type is enabled, specifies the delay in milliseconds before a new query is triggered when the user types new text inside the query box.
The default value is 2000 milliseconds.

Markup configuration example(s) :
data-search-as-you-type-delay='10'
