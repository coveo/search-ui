Coveo component QuerySummary

This component displays information about the current range of results being displayed (ex: 1-10 of 123).
If the query matches no document, it will display advices and tip for the end user on how to remedy the problem.

Component Options

enableSearchTips

enableSearchTips: boolean
  • Defined in src/ui/QuerySummary/QuerySummary.ts :34

Specifies whether the search tips are displayed to the end user when there are no search results.
The default value is true.

Markup configuration example(s) :
data-enable-search-tips='true'
data-enable-search-tips='false'

onlyDisplaySearchTips

onlyDisplaySearchTips: boolean
  • Defined in src/ui/QuerySummary/QuerySummary.ts :39

Specifies whether to hide the information about the current range of results being displayed and only display the search tips.
The default value is false.

Markup configuration example(s) :
data-only-display-search-tips='true'
data-only-display-search-tips='false'
