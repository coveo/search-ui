Coveo StarRating Component (CoveoStarRating)

The StarRating component renders a five-star rating widget for use in commerce result templates.

Available since

January 2020 Release (v2.7968)

Result template component

This component is intended to be used in result templates

Component Options

numberOfRatingsField

numberOfRatingsField: IFieldOption
  • Defined in ui/StarRating/StarRating.ts :53

A numeric field whose value should be used to display the total number of ratings label for the result item.

If unspecified, no number of ratings label is displayed. If the numberOfRatingsField's value is 0 or less, a (No Ratings) label is displayed.

Markup configuration example(s) :
data-number-of-ratings-field='@foo'

ratingField

ratingField: IFieldOption
  • Defined in ui/StarRating/StarRating.ts :46

Specifies the rating to be displayed as stars. If the rating is on a different scale than 0-5, a ratingScale value must be provided.

Required

Specifying a value for this option is required for the component to work

Markup configuration example(s) :
data-rating-field='@foo'

ratingScale

ratingScale: number
  • Defined in ui/StarRating/StarRating.ts :60

The scale to apply to the ratingField's value. Must be smaller than or equal to the highest possible ratingField's value.

Example: If the ratingScale is 100 and the current ratingField's value is 75, the component will render 3 stars (i.e., 75 * (5 / 100), rounded down).

Minimum

1

Maximum

100000

Default

5

Markup configuration example(s) :
data-rating-scale='10'
